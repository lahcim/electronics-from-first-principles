# Volume 5 — Transistors: BJTs and MOSFETs

This volume develops transistor intuition from device physics through switching, amplification, biasing, thermal design, safe operating area, and practical component selection.

It contains 60 lessons in five parts:

1. BJT fundamentals and switching — Lessons 1–18
2. BJT amplifiers and building blocks — Lessons 19–30
3. MOSFET fundamentals and switching — Lessons 31–47
4. MOSFET power design and amplifiers — Lessons 48–57
5. CMOS foundations and capstones — Lessons 58–60

Every lesson includes physical intuition, equations, KiCad 10/ngspice setup, exact directives where required, prediction, experiments, observations, common mistakes, and a design challenge with validation targets.

This is the manuscript-level first draft. A later hardening pass should add one rendered schematic and one native KiCad 10 project per lesson, verified models, waveform data, and generated PDFs.
