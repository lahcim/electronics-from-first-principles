# Volume 2 — Capacitors, Inductors, and Time-Domain Circuits

This volume develops intuition for energy-storage components and changing signals. It contains 40 lessons organized into four parts: capacitor fundamentals, real capacitors, inductors and magnetic energy, and first-order design applications.

Every lesson follows the course pattern: physical intuition, equations, KiCad 10 construction, required SPICE directives, prediction, simulation experiments, observations, common mistakes, design challenge, and validation guidance.

## Parts

1. Capacitor fundamentals — Lessons 1–10
2. Real capacitors and practical selection — Lessons 11–20
3. Inductors, magnetic energy, and coupled components — Lessons 21–35
4. First-order design applications and capstones — Lessons 36–40

## KiCad compatibility

Projects use importable KiCad legacy schematics plus `.kicad_pro` wrappers. Open each `.sch` in KiCad 10, allow conversion, and save as `.kicad_sch`. Required directives are listed explicitly in each lesson and must be marked as SPICE directives rather than ordinary schematic text.
