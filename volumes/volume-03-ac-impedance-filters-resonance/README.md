# Volume 3 — AC Signals, Impedance, Filters, and Resonance

This volume teaches how circuits behave when voltage and current vary with time and frequency. It moves from sine waves and RMS values through complex impedance, Bode plots, passive filters, resonance, damping, and practical filter design.

The volume contains 35 lessons in four parts:

1. AC signals, RMS, phase, and complex numbers — Lessons 1–9
2. Impedance and frequency response — Lessons 10–18
3. Passive filters and loading — Lessons 19–27
4. Resonance, damping, and design capstones — Lessons 28–35

Every lesson includes physical intuition, equations, KiCad 10/ngspice setup, exact SPICE directives when needed, prediction, experiments, observations, common mistakes, and a design challenge with validation targets.

## KiCad compatibility

The first draft is manuscript-level and includes reusable ngspice reference circuits. A later hardening pass should add one rendered schematic and one round-trip-tested native KiCad 10 project per lesson, reference plots, CSV results, and generated PDFs.
