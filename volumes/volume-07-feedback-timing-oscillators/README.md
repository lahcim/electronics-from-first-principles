# Volume 7 — Feedback, Timing, and Oscillators

This volume develops oscillation from feedback fundamentals through RC, LC, relaxation, 555, crystal, PLL, and clock-distribution circuits.

It contains 46 lessons in five parts:

1. Feedback and oscillator foundations — Lessons 1–8
2. RC, relaxation, multivibrator, and 555 timing circuits — Lessons 9–20
3. LC and transistor oscillators — Lessons 21–25
4. Crystal oscillators and precision timing — Lessons 26–40
5. PLLs, jitter, clock distribution, and capstones — Lessons 41–46

Every lesson includes physical intuition, equations, KiCad 10/ngspice setup, required directives, prediction, experiments, observations, common mistakes, and a design challenge with validation targets.

This is the manuscript-level first draft. A later hardening pass should add rendered schematics, native KiCad 10 projects, verified device/crystal models, reference waveforms and spectra, CSV measurements, and generated PDFs.
