# Lesson 3 Design Challenge — Worked Solution

Use two independent ideal sources referenced to node `0`:

- VA from node `A` to node `0`, value `12 V`;
- VB from node `0` to node `B`, value `5 V`, so node `B` is `−5 V`.

Then:

$$V(A,B)=V(A)-V(B)=12-(-5)=17\text{ V}$$

Expected operating-point results:

| Measurement | Result |
|---|---:|
| `V(A)` | +12 V |
| `V(B)` | −5 V |
| `V(A,B)` | +17 V |
| `V(B,A)` | −17 V |

An equivalent valid solution may use series sources, provided the required nodes remain referenced and no source loop with contradictory ideal voltages is created.
