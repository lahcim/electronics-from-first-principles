# Lesson 5 KiCad 10 project

Open `lesson-05.kicad_pro`, then open `lesson-05.sch`. KiCad 10 will import the legacy schematic; save it as `lesson-05.kicad_sch`.

In **Inspect → Simulator**, configure a DC sweep of V1 from 0 V to 10 V in 0.1 V steps. Plot `V(VIN)`, current through R1, and R1 power.

No schematic SPICE directive is required for this lesson. Expected current is 0–10 mA for R1 = 1 kΩ.
