# Lesson 9 KiCad 10 project

Open `lesson-09.kicad_pro`, then open `lesson-09.sch`. Let KiCad 10 import and save it as `lesson-09.kicad_sch`.

Run a DC operating-point analysis. No schematic SPICE directive is required.

Expected baseline:

- `V(VIN)` = 12 V across every branch;
- R1 current = 12 mA;
- R2 current = 6 mA;
- R3 current = 4 mA;
- source-current magnitude = 22 mA;
- equivalent resistance ≈ 545.45 Ω.

Translate simulator current signs into the arrows used in the lesson.
