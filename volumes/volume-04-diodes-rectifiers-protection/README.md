# Volume 4 — Diodes, Rectifiers, Protection, and Semiconductor Junctions

This volume introduces semiconductor junctions and develops practical diode circuits from first principles. It moves from material behavior and diode models through rectification, clipping, clamping, voltage multiplication, flyback protection, reverse-polarity protection, TVS selection, and protected sensor interfaces.

The volume contains 34 lessons in four parts:

1. Semiconductor and diode fundamentals — Lessons 1–9
2. Diode families and switching behavior — Lessons 10–16
3. Rectifiers, smoothing, and wave shaping — Lessons 17–27
4. Protection circuits and capstones — Lessons 28–34

Every lesson includes physical intuition, equations, KiCad 10/ngspice setup, exact SPICE directives where required, prediction, experiments, observations, common mistakes, and a design challenge with validation targets.

## KiCad compatibility

This first draft is manuscript-level and includes reusable ngspice reference circuits. A later hardening pass should add one rendered schematic and one round-trip-tested native KiCad 10 project per lesson, reference plots, CSV results, and generated PDFs.
