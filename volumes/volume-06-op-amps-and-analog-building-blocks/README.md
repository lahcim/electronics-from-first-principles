# Volume 6 — Operational Amplifiers and Analog Building Blocks

This volume develops operational-amplifier intuition from differential gain and feedback through practical limitations, stability, noise, active filters, precision rectification, current sources, transimpedance amplifiers, comparators, and sensor front ends.

It contains 44 lessons in five parts:

1. Op-amp foundations and basic feedback circuits — Lessons 1–12
2. Real op-amp limitations — Lessons 13–26
3. Active filters and signal-processing blocks — Lessons 27–35
4. Comparators, references, and current/photodiode interfaces — Lessons 36–42
5. Design capstones — Lessons 43–44

Every lesson includes physical intuition, equations, KiCad 10/ngspice setup, exact directives when required, prediction, experiments, observations, common mistakes, and a design challenge with validation targets.

This is the manuscript-level first draft. A later hardening pass should add one rendered schematic and one native KiCad 10 project per lesson, verified macromodels and pin mappings, reference plots, waveform data, and generated PDFs.
