# Electronics from First Principles

A simulation-centered electronics course built around physical intuition, mathematical reasoning, KiCad/ngspice experiments, design exercises, and reviewable reference solutions.

The course is being developed incrementally. Each lesson will include:

- physical intuition and a mathematical model;
- a rendered schematic and editable KiCad project;
- predictions to make before simulation;
- parameter experiments and expected observations;
- explanations of why each waveform or measurement occurs;
- common mistakes and model limitations;
- a design challenge with acceptance criteria;
- a separately located worked solution and validation data.

## Current status

The repository foundation and Volume 1 pilot are under development.
